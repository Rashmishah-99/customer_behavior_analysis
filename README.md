## 📌 Overview
This project is an end-to-end **data analytics pipeline** that transforms raw data into meaningful business insights. It includes data cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard development, and professional reporting.

The objective is to demonstrate real-world data analyst workflow using Python, SQL, and Power BI.

## 🚀 Project Workflow

### 1. Data Loading
- Imported dataset using Python (Pandas)
- Checked shape, data types, and missing values

### 2. Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized column names
- Fixed incorrect data types

### 3. Exploratory Data Analysis (EDA)
- Performed statistical summary
- Identified patterns, trends, and anomalies
- Created visualizations (bar charts, histograms, heatmaps)

### 4. SQL Analysis
- Imported cleaned data into SQL database
- Performed:
  - Aggregations (SUM, AVG, COUNT)
  - Grouping and filtering
  - Joins and subqueries
- Extracted key business insights

### 5. Power BI Dashboard
- Built interactive dashboard
- Added KPIs and charts
- Used filters and slicers for dynamic insights

### 6. Reporting
- Summarized insights in a structured report
- Converted technical findings into business insights

### 7. Presentation (Gamma AI)
- Created professional PPT using Gamma
- Included visuals, insights, and project workflow
